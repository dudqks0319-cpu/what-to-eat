# what-to-eat 사업화/마케팅 실행안 (배민 Lite 전략)

## 1) 현재 미완성 원인 요약

- 추천 UX는 있지만 **주문/결제/매장 운영** 흐름이 없음
- CI/품질 게이트 부재(최근까지 lint 실패 상태)
- 사용자 잔존을 만드는 장치(로그인, 알림, 리워드)가 약함

## 2) 제품 전략: "배민 Lite"로 단계적 확장

### Phase 1 (2주): 추천앱 고도화
- 개인화 추천(시간대/예산/최근 선택 기반)
- 결과 화면에 "근처 매장" + "예상 가격" + "배달/포장" 표시
- KPI: 추천→매장 클릭률 15%

### Phase 2 (4주): 주문 연결
- 카카오맵/네이버지도/배달앱 딥링크 연결
- 제휴 링크(쿠팡이츠/요기요/배민) 기반 전환 추적
- KPI: 클릭→주문 전환율 2~3%

### Phase 3 (6~8주): 수익화
- 스폰서 매장 슬롯(지역/시간대 타겟팅)
- 프리미엄(광고 제거, 고급 필터, 팀 점심 추천)
- KPI: ARPU 월 1,000~2,000원

## 3) 수익모델 우선순위

1. **CPA 제휴 수익**: 주문 전환 수수료
2. **광고(스폰서 노출)**: 지역 식당 프로모션
3. **B2B 구독**: 팀 점심 추천 + 월간 통계 리포트

## 4) 마케팅 실행 (첫 4주)

### 타겟
- 20~39세 직장인/자취생
- 반복 고민 상황(점심, 저녁, 야식)

### 채널
- 인스타 릴스/틱톡 쇼츠: "3초 메뉴결정"
- 네이버 카페/블로그: 지역별 점심 추천 포스팅
- 당근/지역 커뮤니티: 로컬 맛집 추천 이벤트

### 캠페인
- Week1: "오늘 뭐먹지 챌린지" (UGC 유도)
- Week2: 회사 점심 조합 테스트(팀 공유 기능 강조)
- Week3: 지역 맛집 제휴 쿠폰 테스트
- Week4: 리타게팅 광고(재방문 유도)

## 5) 운영 지표 대시보드

- 방문자수(DAU/WAU)
- 추천 완료율
- 매장 클릭률
- 주문 전환률
- 재방문율(D7)
- 사용자당 평균 수익(ARPU)
